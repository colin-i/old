Sound Recorder

Sound Recorder is simple tool for recording the sound from the default input wave device.
It is a 32 bit version for Windows.
It tries to detect the 'CD Quality' mode at the driver (11025Hz samples per seccond, 16 bits per sample and 1 channel) and when the start command is received it will capture until another key interrupts the process.
The file is saved as wav. The file is placed in the same folder and with the name: seconds since Jan 1, 1970 + extension.
