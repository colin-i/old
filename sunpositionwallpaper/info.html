Sun Position Wallpaper 1.14

Sun Position Wallpaper is a program to change the desktop backround according to the sun position: morning, day, evening and night.
The latitude, longitude and timezone are required for calculations.
The times are taken from an online php script with the following format: time(),HH:MM,HH:MM,HH:MM,HH:MM
The hour and minutes are taken with the following php function:

<?php
$t=time();
$zenith_rise_set=90+50/60;
$zenith_ctw_start_end=96;
//civil twilight start
date_sunrise($t,SUNFUNCS_RET_STRING,$_GET['lat'],$_GET['long'],$zenith_ctw_start_end,$_GET['offset']);
//sun rise
date_sunrise($t,SUNFUNCS_RET_STRING,$_GET['lat'],$_GET['long'],$zenith_rise_set,$_GET['offset']);
//sun set
date_sunset($t,SUNFUNCS_RET_STRING,$_GET['lat'],$_GET['long'],$zenith_rise_set,$_GET['offset']);
//civil twilight end
date_sunset($t,SUNFUNCS_RET_STRING,$_GET['lat'],$_GET['long'],$zenith_ctw_start_end,$_GET['offset']);
?>
