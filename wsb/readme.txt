Web Site Builder 1.0.2 is a tool to have an overview for web pages, templates and page parts.
Place the templates,pages and pages part in a folder.
Call the program with the map file the only argument.(ex: wsb files.txt)
The first row is the source folder. The second row is the destination folder.
The rest of the file is the site map.
The program will do "filesystem.log" file at the executable folder where "D" is a folder change/make and "F" is a file open in write mode.
